# SmartBot - AI Chatbot | CodeAlpha Internship

A smart AI-powered chatbot with web search integration, built as part of the **CodeAlpha Internship Program** (Task 4).

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Flask](https://img.shields.io/badge/Flask-Web_Server-green?logo=flask)
![License](https://img.shields.io/badge/License-MIT-yellow)

## Features

- **Pattern Matching** - Instant replies for greetings, math, time/date
- **Wikipedia Search** - Answers knowledge-based questions from Wikipedia
- **Web Search** - Searches DuckDuckGo for anything else (like Google!)
- **3D Professional UI** - Dark glassmorphism theme with 3D effects
- **Responsive Design** - Works on desktop and mobile

## Key Concepts Used

`if-elif` | `functions` | `loops` | `input/output` | `API calls` | `error handling`

## Project Structure

```
CodeAlpha_chatbot/
├── app.py                  # Flask web server
├── chatbot_engine.py       # 3-layer smart chatbot brain
├── requirements.txt        # Python dependencies
├── templates/
│   └── index.html          # Chat UI (3D design)
└── static/
    ├── style.css           # Premium dark theme
    └── script.js           # Frontend logic
```

## How to Run

```bash
# 1. Clone the repo
git clone https://github.com/nouranglal810-png/codealpha_chatbot.git
cd codealpha_chatbot

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start the server
python app.py

# 4. Open in browser
# http://localhost:5000
```
